# UI/UX System Notes

## Visual Direction
- Clean, professional, high-contrast dark UI.
- Focused information hierarchy with clear CTA and compact panels.

## Design Tokens
- Spacing scale: `--space-1` to `--space-7`.
- Color tokens: `--bg`, `--panel`, `--text`, `--muted`, `--accent`.
- Radius token: `--radius`.

## Typography
- Primary system stack: `"Segoe UI", "Inter", sans-serif`.
- Restrained usage with heading/body contrast and readable line lengths.

## Responsiveness
- Desktop: two-column productivity layout.
- Tablet/mobile: collapses to single-column.
- Inputs/buttons designed for touch targets and keyboard focus.

## Motion
- Subtle entrance animation (`floatIn`) for hero section.
- Lightweight button hover transitions.
- Motion used for guidance only; not decorative overload.

## Accessibility
- High-contrast foreground/background combinations.
- Focus-visible states with distinct border + halo.
- Semantic forms and labels for daily-log inputs.
